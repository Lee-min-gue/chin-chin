html.to.design 사용 방법

  1. 1. 각 HTML 파일을 브라우저에서 엽니다
  2. 2. Figma에서 html.to.design 플러그인을 실행합니다
  3. 3. 브라우저 URL을 입력하고 Import 합니다
  4. 4. 모든 화면을 Figma 프레임으로 변환 완료!

각 화면은 390px (iPhone 14) 기준으로 제작되었습니다.
모바일 퍼스트 PWA 디자인에 최적화되어 있습니다.
